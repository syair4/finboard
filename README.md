# FinBoard — Stocks & Crypto Dashboard

A modern dark-themed stock and cryptocurrency dashboard built with Next.js 15 and Tailwind CSS.

## Features

- **Market Dashboard** — Real-time stock and crypto prices with sparkline charts
- **Portfolio Tracker** — View holdings, cost basis, and P&L (requires login)
- **Investment Calculator** — Compound interest, ROI, and tax calculators
- **Authentication** — Client-side login/signup flow

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
